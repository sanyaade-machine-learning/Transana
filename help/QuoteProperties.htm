Quote Properties

Please note that the Standard version of Transana does not support Quotes. To use this feature, you must be using the Professionsal, Computer Lab, or Multi-User version of Transana.

A Quote in Transana is a basic analytic unit for text. It is a small segment of text taken from a Document. The Quote is selected as analytically important and is coded by placing it in a Collection and by adding Keywords to it.

There are two types of Quotes in Transana, Standard Quotes and Quick Quotes. Please see the linked pages in the Tutorial for further information about the similarities and differences between these types of Quotes.

There are several ways to trigger the creation of a Quote:

  • Indicate the contents you want included in the Quote:

    • If you do not make a selection in the Document text, Transana will automatically select the sentence surrounding the current cursor position for the Quote text.

    • - or - If you make a selection in the Document text, Transana will use the highlighted text as the Quote text.

  • To create a Standard Quote:

    • You can double-click the target Collection. Doing this will add the Quote to the end of the list of data objects in the Collection.

    • - or - You can right-click (Ctrl-click on the Mac) on the Collection where you want to place the Quote and select "Add Quote" from the popup menu. Doing this will add the Quote to the end of the list of data objects in the Collection.

    • - or - You can drag the selected text from the Document and drop it on a Collection on the database tab of the data window. Doing this will add the Quote to the end of the list of data objects in the Collection.

    • - or - You can right-click (Ctrl-click on the Mac) on an existing Quote, Clip, or Snapshot in a Collection and select "Add Quote" from the popup menu. Doing this will add the Quote to the list of data objects in the Collection just above the selected item.

    • - or - You can drag the selected text from the Document and drop it on an existing Quote, Clip, or Snapshot in a Collection on the database tab of the data window. Doing this will add the Quote to the list of data objects in the Collection just above the selected item.

  • - or - To create a Quick Quote:

    • You can double-click the a Keyword.

    • - or - You can right-click (Ctrl-click on the Mac) on the Keyword and select "Add Quick Quote" from the popup menu.

    • - or - You can drag the selected text from the Document and drop it on a Keyword on the database tab of the data window.

If you are creating a Standard Quote, you will then see the Quote Properties dialog box. (Skipping this dialog box is what makes a Quick Quote quick!)

The only required field is the "Quote ID," which must be unique within the parent Collection. "Comment" can be used in any way you wish. By selecting a Keyword group and Keyword, then clicking the  button, you can add as many analytic keywords as you wish to each Quote, enabling you to find this Quote later when doing keyword searches. The  button allows you to delete keywords from the list, and the  button allows you to access Keyword Management to edit your keywords in the middle of creating a Quote. The "Collection ID," "Library ID," "Document ID," "Quote Start Character," "Quote End Character," and "Quote Length" fields are all maintained by Transana and cannot be edited. While you can edit the Quote text if you like, it is not recommended. Changes made here are not reflected in the original Document text.

When you edit a Quote using Quote Properties from the Quote's right-click menu, the  button will be visible. This button is used to locate copies that match the original Quote ID, source Document number, Quote start character, and quote end character, and to update those copies so that they are identical to the current Quote. See Propagate Quote and Clip Changes for more information.

One final note: Unlike other branches of the database tree, items within a Collection have an "order" that can be altered by the user. (In some forms of Qualitative analysis, such as Narrative Analysis, the order of data objects can be very important. This order is reflected in the Collection Report.) If you choose "Add Quote" or drop your text on a Collection name, your new Quote will be added to the end of the list of Quotes, Clips, and Snapshots within the Collection. However, if you choose "Add Quote" or drop your text on a Quote, Clip, or Snapshot, the new Quote will be inserted in the list above the item where the text is dropped. The order of Quotes, Clips, and Snapshots within a Collection can be altered using cut-and-paste or drag-and-drop within the Collection.
