Document Properties

Please note that the Standard version of Transana does not support Documents. To use this feature, you must be using the Professionsal, Computer Lab, or Multi-User version of Transana.

A Document in Transana is where original source text data has been imported. Once imported, Transana maintains a separate copy of this text data, so changes to the original document file do not affect the Document in Transana and changes to the Transana Document text do not affect the original imported file.

To add a Document in Transana, right-click (Ctrl-click on the Mac) a named Library in the Data window and choose "Add Document." You will see the dialog box shown above. The only required information is an "Document ID" which must be unique within the Library. If you want to import an existing text file, select an "RTF/XML/TXT File to import" using the associated "Browse" button. Othewise the Document will be blank and available for editing. "Author" can be used to indicate the author of the text or the name of the person who imported the Document. "Comment" can be used in any way you wish. The "Library ID" and the "Length" field is filled in automatically by Transana and cannot be edited.

By selecting a Keyword Group and Keyword, then clicking the  button, you can add as many analytic keywords as you wish to each Document. These Keywords are default codes, and are optionally applied to all Quotes created from the Document. Selecting a Keyword Group : Keyword pair in the Keywords list and pressing  removes a keyword from the list, and pressing  allows you to access Keyword Management, which allows you to edit your Keywords while in the middle of creating a Document.
