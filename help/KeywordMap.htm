Keyword Maps

Document Keyword Map

The Document Keyword Map presents a visual display of where Keywords have been assigned to Quotes created from the selected Document. Each colored band in the Document Keyword Map shows the start and stop points of a Quote along the horizontal axis of character position in the Document, with Keywords along the vertical axis. When configured so that color represents Quotes, bands of the same color with the same starting and ending points indicate that a single Quote was assigned multiple Keywords. A tri-color band indicates where multiple Quotes have been coded with the same Keyword at the same character position.

If you hold your cursor over a band in the Document Keyword Map, you will see a popup hint that indicates the name and length of the Quote that the band represents. Left-clicking on a band will load the appropriate Quote in Transana and close the Document Keyword Map. Ctrl-left-clicking on Windows / Cmd-left-clicking on OS X will load the Clip in the context of the Document and close the Document Keyword Map. Right-clicking a band will load the Quote, leaving the Document Keyword Map open but moving it to the background, and Ctrl- or Cmd-right-clicking will load the Quote in context without closing the Document Keyword Map.

The Document Keyword Map has several options in its toolbar (and the menu on Windows only.) You can customize the content using the Filter Dialog. You can save a copy of the Document Keyword Map diagram in JPEG format. The JPEG image created will have the same width and at least the same height as the Document Keyword Map window on the screen, so you may want to resize the window prior to saving your image. Similarly, the dimensions of the window on screen affect the overall shape of the printout when Print Preview or Print is selected from the toolbar or menu. Some experimentation may be required to get the exact image size and proportions you desire in a saved graphic or a printout.

Episode Keyword Map

The Episode Keyword Map presents a visual display of where Keywords have been assigned to Clips created from the selected Episode. If Snapshots have been associated with particular spots in the Episode, the Snapshot coding is also displayed. Each colored band in the Episode Keyword Map shows the start and stop points of a Clip or Snapshot association, along the horizontal timeline of the Episode, where a particular Keyword, along the vertical axis, has been applied. When configured so that color represents Clips, bands of the same color with the same starting and ending points indicate that a single Clip or snapshot was assigned multiple Keywords. A tri-color band indicates where multiple Clips or Snapshots have been coded with the same Keyword at the same time period.

If you hold your cursor over a band in the Episode Keyword Map, you will see a popup hint that indicates the name and length of the Clip or Snapshot that the band represents. Left-clicking on a band will load the appropriate Clip or Snapshot in Transana and close the Episode Keyword Map, while right-clicking a band will load the Clip or Snapshot, leaving the Episode Keyword Map open but moving it to the background.

The Episode Keyword Map has several options in its toolbar (and the menu on Windows only.) You can customize the content using the Filter Dialog. You can save a copy of the Episode Keyword Map diagram in JPEG format. The JPEG image created will have the same width and at least the same height as the Episode Keyword Map window on the screen, so you may want to resize the window prior to saving your image. Similarly, the dimensions of the window on screen affect the overall shape of the printout when Print Preview or Print is selected from the toolbar or menu. Some experimentation may be required to get the exact image size and proportions you desire in a saved graphic or a printout.

Collection Keyword Map

The Collection Keyword Map provides a graphical display of the coding for all Clips and Snapshots that have been gathered together into a Collection. At this time, Quotes are NOT included in the Collection Keyword Map. Since Clips can be taken from many different Episodes, Transana constructs an artificial time line for this report by placing the Clips and Snapshots next to each other. Snapshots without a defined duration are assigned a duration of 10 seconds in this time line. Whether this time line is meaningful and can be interpreted depends upon whether the order of the Clips in your Collection is meaningful.

The Collection Keyword Map can be customized in a variety of ways using the filter dialog. It can be saved in JPEG format and printed. In addition, if you click a band in the report, the associated clip will be loaded in Transana.
